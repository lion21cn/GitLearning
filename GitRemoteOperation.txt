远征仓库操作:

1.<远程地址>  从远程仓库克隆
git clone 

2.将最新的提交从远程仓库抓取下来
git pull 

3. 将本地修改后的代码提交到远程仓库
git push 

git push <远程仓库名，默认origin> <本地分支名>  将指定的分支推送到远程分支上

4.查看远程仓库  -v 为详细信息

git remote -v 

git checkout -b <本地支分支名> <远程仓库名，默认origin>/<远程支分支名> 拉取远程主分支下的支分支。。。
git branch --set-upstream <本地支分支名> <远程仓库名，默认origin>/<远程支分支名>  将本地分支与远程指定的分支关联起来
